Description

Creating a banking application involves several stages, and the first one is registration. In this stage, we collect user data and store it for re-checking before the final registration. We must obtain the data and retain it securely for future verification. Only after our users enter their username and password again without errors, the registration is successful! The program will congratulate the user upon successful account registration and end.

Objectives

In this stage, your program should satisfy the following conditions:

  • Ask for their username by printing Username:, then ask for their password: Password: and save both values.
  • After this, print an empty line to make your output more readable.
  • After successfully entering both username and password, prompt to enter the data again by printing: Enter the data again to complete the registration.
  • Repeat Step 1.
  • Compare the first data with the second. If the usernames or passwords do not match, print an error message. Password error Passwords don't match. Username error: Username doesn't match. If both options are not correct: Username and password don't match. If both usernames and passwords are the same, print the message after the empty line Congratulations on your successful registration!
  • End the program in case of a data entry error or successful registration.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
